--- layout: post title: "[M\xC3\xA9mo] Installation de XDebug sous ubuntu" published: true meta: _edit_last: "1" tags: - "D\xC3\xA9faut" - netbeans - php - xdebug type: post status: publish --- Sous debian, par les paquets : 
sudo aptitude install php5-xdebug
apache2ctl restart
XDebug devrait être intégré au prochain redémarrage d'apache. Les messages d'erreurs devraient maintenant être un peu plus bavards. Pour activer le débugage pas à pas dans netbeans (ou autre) : 
echo "xdebug.remote_enable=on" >> /etc/php5/apache2/conf.d/xdebug.ini
apache2ctl restart
Il ne reste qu'à ajouter des points d'arrêt dans netbeans et faire Ctrl+F5
